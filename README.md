# 📝 To-Do List App (Python Tkinter)

This is a simple and professional To-Do List application built using Python and Tkinter.

## 🚀 Features
- Add tasks
- Delete tasks
- Mark tasks as completed ✔
- Save tasks to file
- Load tasks automatically
- Clean and user-friendly interface

## 🛠️ Technologies Used
- Python
- Tkinter

## ▶️ How to Run
1. Open the file in Python (Thonny or any IDE)
2. Run the file:
   python todo_gui.py

## 📂 Files
- todo_gui.py → Main application
- tasks.txt → Stores saved tasks
